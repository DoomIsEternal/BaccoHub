# BaccoHub

BaccoHub is an open source hub aimed at helping new exploiters learn aswell as providing an alternative to many payed hubs such as WallyHub, KiriotHub etc.

## Support

BaccoHub supports the most popular paid exploits such as Synapse X, ProtoSmasher and Sentinel.
